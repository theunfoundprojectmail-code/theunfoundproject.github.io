# The Unfound Project Website

This is a static GitHub Pages-ready starter site for The Unfound Project.

## Included pages

- `index.html` — homepage
- `framework.html` — named project model / framework page
- `book.html` — book landing page
- `about.html` — project description
- `cases/index.html` — case archive landing page
- `cases/jason-jolkowski.html`
- `cases/brian-shaffer.html`
- `cases/jennifer-kesse.html`

## How to publish on GitHub Pages

1. Create a GitHub repository named `theunfoundproject.github.io`.
2. Upload all files from this folder to the repository root.
3. Commit the files.
4. In GitHub, open **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/ (root)` folder.
7. Save.

Your site should publish at:

`https://theunfoundproject.github.io`
